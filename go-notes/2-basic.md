# Go语言基础
### 包结构
- go工作空间
  - src
    - proj1
      - module1
      - module2
    - proj2
  - bin
    - 编译后的二进制文件
  - pkg
    - 编译后的库文件

---

### 执行流程
- go build
编译go源码->生成可执行文件
- go run
直接运行源代码

### 字符串操作
- fmt格式化输出
- strconv解析字符串，类型转换
- 直接拼接

### 内存分配与指针
- 基本数据类型直接在栈上分配内存
- 引用类型实际对象在堆上分配内存

### 运算符注意事项
- 自增自减只能出现在变量后面，并且只能单独出现(不可用于赋值)
- go中不支持三元运算符

### 进制表示
- 不支持直接表示二进制
- 八进制由0开头
- 十进制默认
- 十六进制由0x或0X开头
